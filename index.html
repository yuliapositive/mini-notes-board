📖 Как пользоваться доской

📌 База
  • ➕ Введите текст, выберите тег → «Добавить»
  • ☑️ Клик по чек-боксу — отметить выполненное
  • 🏷️ Каждый тег получает свой цвет
🎨 Оформление
  • 🎨 Нажмите 🎨 — выберите цвет карточки
  • 📌 Нажмите 📌 — закрепите сверху
  • 🔒 Нажмите 🔒 — защитите PIN-кодом
  • 📎 Перетащите файл на заметку — добавьте вложение
⚡ Функции
  • ⏰ Нажмите ⏰ — установите напоминание
  • 🔗 Ссылки становятся кликабельными автоматически
  • 🎙️ Нажмите 🎙️ — диктуйте текст
  • 📜 Нажмите 📜 — откатите версию
⌨️ Управление
  • 🔍 Ctrl+K — поиск
  • ⌨️ ESC — закрыть, ENTER — добавить
  • ⌨️ Ctrl+Z — отменить, Ctrl+Y — повторить
  • 👤 Зарегистрируйтесь → заметки в аккаунте
