# 📌 Доска заметок v1.9.0

> Минималистичная, удобная, твоя. Доска заметок с умными тегами, графиком активности, календарём и ИИ-помощником.

---

## 🚀 Демо

**[Живая демонстрация](https://yuliapositive.github.io/mini-notes-board/)**

---

## ✨ Особенности

### 🏷️ Умные теги
- Пиши `#тег` прямо в тексте заметки
- Каждый тег получает свой уникальный цвет
- Клик по тегу — фильтрация заметок
- Автодополнение тегов при вводе

### 📊 График активности
- Визуализация 30 дней активности
- Тепловая карта: от серого к фиолетовому
- При наведении — дата и количество заметок

### 📅 Календарь
- Просмотр заметок по дням
- Навигация по месяцам
- Дни с заметками подсвечиваются

### 🔍 Расширенный поиск
- Поиск по тексту
- Фильтр по тегу: `#работа`
- Фильтр по дате: `@2026-07-23`
- Фильтр по статусу: `done:false`

### 📎 Вложения
- Drag & Drop на карточку заметки
- Превью изображений
- Прогресс-бар загрузки

### 🔔 Напоминания
- Установка даты и времени
- Повторение: ежедневно, еженедельно, ежемесячно
- Push-уведомления

### 📤 Экспорт
- JSON (полный бэкап)
- Markdown (.md)
- Plain Text (.txt)
- **DOCX** (.docx) — новый!
- **PDF** (.pdf) — новый!
- Google Docs

### 📋 Шаблоны
- Список дел, Идея, Заметка, Чек-лист, План на день
- Сохранение своих шаблонов
- Быстрый доступ

### 🎨 Оформление
- Тёмная / светлая тема
- Авто-тема (по системе)
- Кастомные темы (создай свою!)
- Звёздный фон с анимацией

### 🔐 Безопасность
- PIN-код для защиты доски
- Защита отдельных заметок PIN-кодом
- Авторизация и аккаунты

### ⌨️ Горячие клавиши
| Комбинация | Действие |
|------------|----------|
| `Enter` | Добавить заметку (на доске) |
| `Enter` | Вернуться на доску (в корзине) |
| `Ctrl+K` | Фокус на поиск |
| `Ctrl+A` | Выбрать все заметки |
| `Ctrl+D` | Удалить выбранные |
| `Ctrl+P` | Закрепить выбранные |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Повторить |
| `ESC` | Закрыть всё |

## 🛠️ Технологии

- **HTML5** — структура
- **CSS3** — стилизация (кастомные свойства, анимации, медиа-запросы)
- **Vanilla JavaScript** — вся логика (без зависимостей!)
- **LocalStorage** — хранение данных
- **PWA** — установка на телефон
- **Web Speech API** — голосовой ввод
