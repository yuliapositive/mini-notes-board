# 📌 Мини-доска заметок 

Минималистичная доска для заметок с продвинутыми функциями.

🌐 **Демо:** [Открыть доску заметок](https://yuliapositive.github.io/mini-notes-board/)

---

## 🚀 Функции

### 🔐 Безопасность
- PIN-код для защиты доски (устанавливается по желанию)
- Защита отдельных заметок 🔒

### ✨ Основные функции
- ✅ Создание/редактирование/удаление заметок
- ✅ Чек-листы с прогрессом
- 📅 Напоминания с уведомлениями
- 🔗 Кликабельные ссылки
- 🏷️ Цветные теги
- 📌 Закрепление заметок

### 🎯 Продвинутые функции 
- 📎 Drag & Drop для вложений
- 🖼️ Превью изображений в карточках
- 📊 Прогресс-бар загрузки
- 📱 PWA — установка на телефон как приложение
- 🔔 Push-уведомления о напоминаниях
- 📊 Расширенная статистика (лучший день, топ-тег, вложения)
- 🎨 Кастомизация тем
- ⌨️ Новые горячие клавиши

### 🎨 Интерфейс
- 🌓 Тёмная/светлая/авто тема
- 🖼️ 2 режима просмотра: плитка/список
- ✨ Анимированный звёздный фон
- 💎 Эффект стекла (Glassmorphism)
- 📱 Полная адаптивность

### 💾 Хранение
- Автосохранение в localStorage
- 📤 Экспорт/импорт JSON, Markdown, TXT
- ☁️ Синхронизация и резервное копирование

---

## ⌨️ Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `ENTER` | Добавить заметку |
| `ESC` | Закрыть всё |
| `Ctrl+A` | Выделить все заметки |
| `Ctrl+D` | Удалить выделенные |
| `Ctrl+P` | Закрепить выделенные |
| `Ctrl+Z` | Отменить действие |
| `Ctrl+Y` | Повторить действие |
| `Ctrl+K` | Фокус на поиск |
| `Ctrl+Shift+R` | Открыть регистрацию |
| `Ctrl+Shift+L` | Заблокировать доску |
| `Ctrl+Shift+P` | Открыть профиль |
| `Ctrl+Shift+C` | Открыть календарь |

---

## 🔐 PIN-код

По умолчанию PIN не установлен.
Установите его на главной странице → кнопка **"🔐 Установить PIN"**

---

## 📦 Установка

Просто откройте `index.html` в браузере!

---

## 📱 PWA

Доску можно установить на телефон как приложение:
- Откройте сайт в Chrome/Safari
- Нажмите **"Добавить на экран 'Домой'"**
- Готово! 🎉
